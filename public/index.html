Recent Projects

Project 1

Fiber Link Installation

I was part of a team responsible for the installation and splicing of fiber optic cables, ensuring high-speed and reliable data transmission for a Morupule Coal Mine's network infrastructure. Implemented industry best practices for cable management and termination, resulting in minimal signal loss and optimal network performance.

Project 2

Network Upgrade and Expansion

Planned the layout of the network cabling, which included cable routing, where to mount trunking, cabinets, and Access Points. Installed and terminated the CAT6 Ethernet cables to surface mount boxes and patch panels using the TIA/EIA 568 B standard which is commonly used in the Morupule coal Mine. Tested and verified the integrity of cables using cable testers. Ensured proper labeling of the cables.

Project 3

CCTV System Integration

Carried out the integration of a CCTV system with an existing security infrastructure, including the installation of cameras, DVRs, and monitoring software at 4Tech It, Acacia Clinic and other companies around Gaborone. Implemented remote access and monitoring capabilities, enhancing the client's ability to monitor their premises effectively.

Project 4

VoIP Implementation

Led the implementation of a Voice over IP (VoIP) system for a small business client, replacing their traditional phone system. Configured IP phones, set up SIP trunking, and integrated the system with their existing network infrastructure. Provided training to staff on using the new system, resulting in improved communication efficiency and reduced monthly telecommunication costs.

Project 5

Wireless Network Deployment

Designed and deployed a comprehensive wireless network solution for a multi-building campus. Conducted site surveys to determine optimal access point placement, implemented wireless security protocols, and configured VLANs for traffic segmentation. The solution provided seamless connectivity across the entire campus while maintaining strong security and performance.

Project 6

Network Security Upgrade

Performed a comprehensive security audit and upgrade for a client's network infrastructure. Implemented firewall rules, intrusion detection systems, and VPN access for remote workers. Configured network segmentation to isolate sensitive data and conducted security awareness training for employees. The project significantly enhanced the organization's security posture and reduced vulnerability to cyber threats.
